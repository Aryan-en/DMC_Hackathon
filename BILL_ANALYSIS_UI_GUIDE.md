## User Workflow

### Step 1: Navigate to Bill Analysis
```
1. Click "Bill Amendment" in Sidebar under COMMAND section
2. Page loads at http://localhost:3002/bill-analysis
```

### Step 2: Upload Document
```
1. Click upload area or drag a PDF file
2. System validates file type (PDF only)
3. File selected: "digital_privacy_act.pdf"
```

### Step 3: Trigger Analysis
```
1. Click "Analyze Bill" button
2. File uploads to /api/bill-analysis/analyze
3. Backend processes PDF and extracts text
4. AI analysis synthesizes results
5. Response returns in JSON format
```

### Step 4: View Results
```
1. Bill Summary appears at top
2. Scroll to view all analysis sections
3. Interact with charts (hover for details)
4. Review all dimensions:
   - Pros and Cons
   - Economic Impact
   - Risk Assessment
   - Global Consequences
   - Implementation Plan
   - Stakeholder Effects
   - Comparative Analysis
```

### Step 5: Download/Export (Future Feature)
```
Future: Export analysis as PDF report
Future: Save analysis to history
Future: Compare multiple bills
```

## Data Metrics Explained

### Economic Indicators
- **GDP Impact**: National economic growth change (-5% to +5%)
- **Employment**: Job creation (+) or job losses (-) in percentage
- **Inflation**: Price level adjustment due to law
- **Sectors**: Individual industry impact (7+ sectors)

### Risk Metrics
- **Risk Level**: LOW / MEDIUM / HIGH classification
- **Probability**: 0-100% likelihood of risk occurring
- **Influence**: 0-100% stakeholder influence score

### Global Metrics
- **Geopolitical Influence**: 0-100% scale of global impact
- **Trade Relations**: Positive/negative bilateral effects
- **Regional Scope**: 1-8 geographic regions affected
